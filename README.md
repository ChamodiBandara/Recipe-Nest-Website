# Recipe Nest 🍲

## 📌 Description

Recipe Nest is a full-stack web application that allows users to explore, create, and manage food recipes through an interactive and user-friendly interface.

---

## 🚀 Features

- 🔐 User authentication (Login & Signup)  
- ➕ Add new recipes  
- ✏️ Edit existing recipes  
- ❌ Delete recipes  
- 🔍 Search recipes  
- 🎯 Filter recipes (by category, ingredients, etc.)  
- 📥 Download recipes  
- 📱 Interactive and responsive UI  

---

## 🛠️ Tech Stack

**Frontend:** React.js

**Backend:** Node.js, Express.js

**Database:** MySQL

---

## 🏗️ Project Structure

* `frontend/` – React application
 
* `backend/` – API and server-side logic

---

## ▶️ How to Run

### 1. Clone the repository

```
git clone https://github.com/YOUR-USERNAME/Recipe-Nest-Website.git
```

### 2. Install dependencies

```
cd frontend
npm install

cd ../backend
npm install
```

### 3. Run the application

```
# Start backend
cd backend
npm install
npm start

# Start frontend
cd frontend
npm install
npm start
```
